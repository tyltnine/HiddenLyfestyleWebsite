# Hidden Lyfestyle website

Standalone HTML/CSS/JavaScript portfolio site created for **Hidden Lyfestyle / @styledbymelody1**.

## Open it

Double-click `index.html`, or serve the folder locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Add the real photos

Put the supplied images in `assets/images/` using these exact names:

- `hero.jpg`
- `look-01.jpg`
- `look-02.jpg`
- `look-03.jpg`
- `look-04.jpg`
- `look-05.jpg`
- `look-06.jpg`

The labeled placeholders disappear automatically when those files exist. Portrait or vertical images work best for most slots. Use web-optimized JPEG or WebP images, ideally 1600–2400px on the long edge.

## Before publishing

1. In `script.js`, replace `booking@hiddenlyfestyle.com` with the actual inquiry email.
2. Confirm service wording, location, pricing language, and client permissions.
3. Add a favicon and social-share image if desired.
4. Only publish photos the stylist/client has permission to use.

## Notes

The visual direction is inspired by the editorial scale and movement of the reference site, but the code, branding, typography system, layout details, and copy were rebuilt specifically for Hidden Lyfestyle rather than copying the original source one-for-one.
